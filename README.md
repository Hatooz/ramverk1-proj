# ramverk1-proj

Sidan använder sig av sqlite3. För att få en egen version, bekräfta att du har sqlite3 och klona repot. Ställ dig sen i rooten och kör kommandot:
sqlite3 data/db.sqlite < sql/ddl/all.sql
